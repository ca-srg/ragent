# OCR 追加プロンプトテンプレート

このファイルは、PDF OCR 用の追加指示を書くためのテンプレートです。リポジトリには `ocr-prompt-template.md` を保持し、実際に使う内容は `ocr-prompt.md` にコピーして管理してください。

## 使い方

1. テンプレートをコピーします。

   ```bash
   cp ocr-prompt-template.md ocr-prompt.md
   ```

2. 必要な追加指示を `ocr-prompt.md` に記述します。

3. `--ocr-prompt-file` で指定して実行します。

   ```bash
   ragent vectorize --ocr-prompt-file ocr-prompt.md
   ```

## 重要な注意

- このファイルの内容は、ベースの OCR プロンプトを**置き換えるのではなく追記**されます。
- ベースの OCR プロンプトは、各ページを要素に持つ **JSON 配列** を返す前提です。出力形式に矛盾する指示を書かないでください。
- Markdown コードフェンスの追加や自由文の説明など、**出力形式を変更する指示は入れないでください**。

## サンプルプロンプト

```md
## 追加指示

以下の点に特に注意してテキストを抽出してください：
- 表やグラフ内のデータも可能な限り正確にテキスト化してください
- ヘッダーやフッターの情報も含めてください
```

## secret フィールドの制御

ベースプロンプトの JSON スキーマには `"secret"` フィールド（boolean、デフォルト `false`）が含まれています。追加プロンプトで条件を指定すると、OCR 結果のメタデータに `secret: true` を付与できます。

```md
## 追加指示

ドキュメントの内容が 1on1 の場合は `secret: true` を設定してください。
```
